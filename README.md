# pos-system
Point of Sale System

A desktop-based Point of Sale (POS) application developed in C# (Visual Studio) 
with SQL Server as the backend database. Designed for efficient transaction 
management with built-in business intelligence tools.

Features
- Point of Sale – Fast and intuitive transaction processing
- SQL Server Integration – Reliable and structured data storage
- Admin Dashboard – Monitor and track sales performance in real time
- Sales Forecasting – Python-powered predictive analytics to forecast future sales trends

Tech Stack
- Frontend/Backend: C# (.NET / WinForms / WPF)
- Database: Microsoft SQL Server (SSMS)
- Forecasting: Python
